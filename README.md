# Luxe Parfum Store

A modern e-commerce site for luxury perfumes built with React, Vite, and Tailwind CSS.

## Features

- Luxury homepage with product grid
- Product listing page with search
- Product detail page with Add to Cart and WhatsApp order button
- Shopping cart with item removal and total
- Checkout page with order confirmation
- Login and Register pages (localStorage-based auth)
- User profile page with logout
- Admin dashboard for adding new products
- Responsive design with Tailwind CSS

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser at `http://localhost:5173`.

## Tech Stack

- React 18
- Vite 4
- Tailwind CSS 3
- React Router DOM 6
- localStorage (cart and auth)
